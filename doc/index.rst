PyPE's (Python Parsing Expressions) Documentation
=================================================

PyPE is a python library for building parsers. It is modeled after the Lua LPEG
library. The syntax has been preserved where possible, although not all of the
operations of LPEG are included, and some additional feattures that are not
part of LPEG are added.
